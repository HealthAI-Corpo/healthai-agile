# Conventions de Travail - healthai-workout

## 1. Branches
- `main` : Stable / Production.
- `develop` : Integration.
- `feat/us-[ID]-[desc]` : Nouvelles fonctionnalités.

## 2. Commits (Conventional Commits)
- `feat:` : Nouvelle fonctionnalité.
- `fix:` : Correction de bug.
- `docs:` : Documentation.
- `chore:` : Maintenance / Outils.

## 3. Workflow
1. Créer une branche depuis `develop`.
2. Déplacer l'issue en "In Progress".
3. Ouvrir une PR vers `develop` une fois fini.
4. Validation par le PO avant merge.
